3. Ví dụ viết công thức Toán trên Web

Đề bài: Viết mệnh đề logic sau bằng HTML.

"Với mọi x thuộc tập số thực R, ta có bình phương của x luôn lớn hơn hoặc bằng 0."

Kết quả hiển thị:

∀x ∈ ℝ, x2 ≥ 0

(Code: &forall;x &in; &Ropf;, x<sup>2</sup> &ge; 0)
